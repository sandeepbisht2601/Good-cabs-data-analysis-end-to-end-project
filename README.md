## 📌 Project Overview

This project is an end-to-end Business Analytics solution built using SQL and Power BI for the fictional cab service company GoodCabs.

The objective of this project is to solve real-world business problems by:

- Analyzing Cab service performance data using SQL
- Generating business insights from raw datasets
- Building an interactive Power BI dashboard for decision-making

The project demonstrates practical skills in:

- Data Analysis
- SQL Query Writing
- Business Problem Solving
- Data Visualization
- Dashboard Development

## 🛠 Tools & Technologies Used

SQL – Data cleaning, transformation, and business analysis
Power BI – Interactive dashboard creation and visualization
Excel/CSV Dataset – Source data

## 📊 Business Problems Solved Using SQL

Some key business questions solved in this project:

- Total rides and revenue analysis
- City-wise trip performance
- Peak and Low Months 
-- Top-performing cities
-- Repeat vs new customer analysis
-- Average fare per trip
-- Monthly growth trends
-- Customer rating insights
-- Revenue contribution analysis
